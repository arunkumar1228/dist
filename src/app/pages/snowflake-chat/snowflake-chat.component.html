What is Qualioptix?
Qualioptix is an innovative application designed to optimize both performance and cost of users' Snowflake accounts. Originally developed as a Snowflake app, Qualioptix recommends techniques to enhance efficiency while minimizing expenses.
How does Qualioptix optimize performance and cost?
Qualioptix offers real-time monitoring of performance metrics and provides detailed cost analysis of each service including warehouses, storage, and cloud services. Based on this analysis, it provides recommendations to optimize performance and reduce costs for each service.
What types of optimization problems can Qualioptix handle?
Qualioptix helps optimize both performance and cost of Snowflake accounts. It recommends optimization techniques at the warehouse, query, storage, and loading process levels within Snowflake

How can I install the app?
  • You can find the app on the Snowflake Marketplace under the Data Products tab. Click "Get", and the app will show in the Apps tab.
  • Then, you can install the app by clicking "Get App."
  • Note: You must use the ACCOUNTADMIN role or another role with the CREATE DATABASE and IMPORT SHARE privileges to access and install a listing.
What privileges are needed for using this app?
You only need Imported privileges on the database named “Snowflake.” Run the following code to grant privileges:
  • GRANT IMPORTED PRIVILEGES ON DATABASE SNOWFLAKE TO APPLICATION <APPLICATION NAME>;
  • Replace <APPLICATION NAME> with your application name. The default application name is QUALIOPTIX.
